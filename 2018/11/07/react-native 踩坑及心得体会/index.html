  • h.config not found ….

  • android 键盘顶开底部导航 (createBottomTabNavigator)

修改 android/app/src/AndroidManifest android:windowSoftInputMode 的值为 stateAlwaysHidden|adjustPan

  • 使用 antd-mobile

npm i 之后 react-native run-ios 会报关于 react-dom 的错误，删除 node_modules 并使用 yarn 即可解决

  • vscode 快速打包 android APK 任务配置

每次打包按下 cmd + shift + p + Run Task + build react-native android apk 一直回车即可

  • 查看文件代码行数

  • 关于 CocoaPods 配置

不知道哪一版本开始(当前0.57.4)，要在 ios/Podfile 里这样写了

  • React-native run-android 模拟器启动失败

    错误1：unable to load script from assets ‘index.android bundle’ ,make sure your bundle is packaged correctly or youu’re runing a packager server

    1.在 android/app/src/main 目录下创建一个 assets空文件夹
    2.在项目根目录运行
    3.重新react-native run-android

    错误2：could not connect to development server. Try the follwing to fix the issue:***

    解决办法：
    重新编译index.js文件(走一遍错误一中的第2部),在重新运行react-native run-android
    双击R键刷新android模拟器
    另外运行ios时，如果想指定某个模拟器
    react-native run-ios –simulator “iPhone X”
    查看iOS设备：在终端中输入
    xcrun simctl list devices

  • Could not connect to development server after updating react-native.

解决办法: add android:usesCleartextTraffic="true" to your application, in the manifest
